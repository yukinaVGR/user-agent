<p>install tool di terminal android / ios, bisa menggunakan terminal 'Termux'. Commands:</p>
<pre><code>
<p>
pkg update && pkg upgrade
pkg install git
pkg install python
git clone https://github.com/yukinaVGR/user-agent
pip install requests
pip install bs4
</p>
</code></pre>

cd user-agent
python run.py
</p>
</code></pre>
